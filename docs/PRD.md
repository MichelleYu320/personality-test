# 프로젝트 개요 및 PRD

## 📋 프로젝트 개요

### 프로젝트명
**"우리 가족 성향 알아보기" - TCI 밸런스 게임**

### 프로젝트 유형
심리 테스트 웹 애플리케이션 (단일 HTML 파일 기반)

### 프로젝트 목적
- 가족 구성원들의 성격 유형을 TCI(Temperament and Character Inventory) 4개 척도를 통해 간편하게 측정
- 밸런스 게임 형식의 재미있는 인터랙션으로 심리 테스트 진행
- 연령대별(10대/20대/30대) 맞춤 솔루션 제공

## 📄 PRD (Product Requirements Document)

### 1. 제품 비전
가족 구성원들이 부담 없이 즐기면서 서로의 성향을 이해할 수 있는 심리 측정 도구

### 2. 핵심 기능

#### 2.1 테스트 구조
- **총 60문항** (4개 파트 × 15문항)
- **4개 측정 척도**:
  - **NS (Novelty Seeking)**: 자극 추구 - 호기심, 모험심, 충동성, 자유분방함
  - **HA (Harm Avoidance)**: 위험 회피 - 조심성, 불안, 신중함, 비관적 사고
  - **RD (Reward Dependence)**: 사회적 민감성 - 공감 능력, 인정 욕구, 감수성, 관계 지향
  - **P (Persistence)**: 인내력 - 끈기, 성취욕, 완벽주의, 승부욕

#### 2.2 사용자 인터페이스

##### 시작 화면
- 테스트 구성 안내
- 4개 파트 개요 표시
- TIP: 3초 안에 선택하도록 가이드
- 테스트 시작 버튼

##### 테스트 화면
- 진행률 바 (실시간 업데이트)
- 진행 상황 텍스트 (예: "15 / 60 문항")
- 현재 파트 표시 (파트명 + 설명)
- 질문 번호 (파트 내 위치)
- 질문 내용
- A/B 2지선다 선택 버튼
- 각 선택지에 대한 상세 설명

##### 결과 화면
- 완료 메시지
- 4개 척도별 점수 시각화 (바 차트)
- 점수 및 퍼센트 표시
- 다시 테스트하기 버튼

#### 2.3 채점 방식
- A 선택 시: 해당 척도 +1점
- B 선택 시: 점수 변동 없음
- 각 척도별 15점 만점 (100%)
- 백분율로 환산하여 결과 표시

#### 2.4 질문 예시 (척도별)

**NS (자극 추구)**
- 새로 생긴 식당 vs 검증된 맛집
- 롤러코스터 vs 회전목마
- 투자 vs 예금
- 즉흥 여행 vs 계획 여행

**HA (위험 회피)**
- 가스밸브 확인 집착 vs 한 번만 확인
- 건강 걱정 과다 vs 낙관적
- 발표 전 불안 vs 태연함
- 미래 걱정 vs 현재 중시

**RD (사회적 민감성)**
- 영화 보고 우는 vs 안 우는
- 공감 vs 해결책 제시
- 인정 욕구 vs 자기 만족
- 거절 못함 vs 거절 잘함

**P (인내력)**
- 끝장내기 vs 중단 가능
- 승부욕 vs 쿨함
- 스케줄 철저 vs 융통성
- 완벽주의 vs 효율성

### 3. 기술 스택
- **프론트엔드**: 순수 HTML/CSS/JavaScript (프레임워크 없음)
- **반응형 디자인**: 모바일/태블릿/데스크탑 대응 (@media query)
- **배포 방식**: 단일 HTML 파일 (별도 서버 불필요)
- **데이터 구조**: JavaScript 객체로 테스트 데이터 관리

### 5. 결과 해석 시스템

#### 유형 분류 체계
- **4비트 코드**: NS-HA-RD-P 각 척도의 높음(A)/낮음(B) 조합
- **16가지 유형**: AAAA, AAAB, AABA, AABB, ABAA, ABAB, ABBA, ABBB, BAAA, BAAB, BABA, BABB, BBAA, BBAB, BBBA, BBBB
- **판정 기준**: 각 척도별 50% 이상 시 A(높음), 미만 시 B(낮음)

#### 4가지 그룹 분류

**GROUP 1. 열정 과다형 (NS↑ HA↑)**
- 특징: 하고 싶은 건 많은데 걱정도 많음
- 유형: AAAA, AAAB, AABA, AABB

**GROUP 2. 직진 본능형 (NS↑ HA↓)**
- 특징: 겁 없는 행동파, 쾌락주의자
- 유형: ABAA, ABAB, ABBA, ABBB

**GROUP 3. 안전 제일형 (NS↓ HA↑)**
- 특징: 신중하고 성실함, 불안 많음
- 유형: BAAA, BAAB, BABA, BABB

**GROUP 4. 태평천하형 (NS↓ HA↓)**
- 특징: 무던하고 멘탈 강함
- 유형: BBAA, BBAB, BBBA, BBBB

#### 각 유형별 제공 정보
- **별명**: 유형을 한마디로 표현
- **특징**: 성격 요약
- **🌱 10대 솔루션**: 학업/친구 관계 조언
- **🎓 20대 솔루션**: 진로/취업 조언
- **💼 30대 솔루션**: 직장/가정 조언

#### 유형별 예시

**AAAA - "번아웃이 온 열정적인 완벽주의자"**
- 모든 척도 최고점
- 슈퍼맨/원더우먼형
- 리스크: 번아웃, 과도한 자책

**BBBB - "유유자적 선비 (나는 자연인이다)"**
- 모든 척도 최저점
- 진정한 자유인
- 리스크: 현실 감각 부족

**ABAA - "에너자이저 핵인싸 (오지랖 대장)"**
- NS↑, HA↓, RD↑, P↑
- 리더십형
- 리스크: 과도한 에너지 소모

### 8. 주요 함수

#### `startTest()`
- 시작 화면 숨김
- 테스트 화면 표시
- 첫 번째 질문 로드

#### `showQuestion()`
- 현재 파트 및 질문 표시
- 진행률 업데이트
- 선택지 렌더링

#### `selectAnswer(choice)`
- 답변 저장 (A 선택 시 해당 척도 +1)
- 다음 질문으로 이동
- 마지막 질문 시 결과 화면 이동

#### `showResult()`
- 테스트 화면 숨김
- 결과 화면 표시
- 4개 척도별 점수 시각화
- 애니메이션 효과

#### `restartTest()`
- 모든 데이터 초기화
- 시작 화면으로 복귀
